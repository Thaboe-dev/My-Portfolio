# Icons Specification

## Overview
- Target file: `src/components/icons.tsx`
- Interaction model: static SVG icons inheriting currentColor.

## Icons
- Social icons from extracted inline SVGs: GitHub, LinkedIn, CodePen, Instagram, Goodreads.
- Utility icons: external arrow, star.

## Styles
- Social icons render at `24px x 24px`, currentColor, hidden text labels for screen readers.
- External arrow renders inline at `12px x 12px`, shifts slightly on parent hover.
- Star renders inline at `12px x 12px`, currentColor.
